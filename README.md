## Описание

Android_fitness — это Android‑приложение для отслеживания тренировок и прогресса пользователя в фитнесе. Проект демонстрирует базовую архитектуру Android‑приложения, работу с экранами, вводом данных и сохранением состояния.

## Возможности

- Создание и редактирование тренировок (упражнения, подходы, повторения, вес).
    
- Отслеживание прогресса по ключевым показателям (повторения, рабочий вес, длительность).
    
- Удобный интерфейс под мобильные устройства на Android.
    

## Технологии

- Язык: Kotlin.
    
- Платформа: Android SDK.
    
- Использование стандартных библиотек и компонентов Android (Activity/Fragment, View, Layout, Intent и т.п.).
  
- Особенность приложения использование Android Speech Recognition API
    

## Установка и запуск

1. Клонировать репозиторий:  
    `git clone https://github.com/VladislavTipa/Adnroid_fitness.git`
    
2. Открыть проект в Android Studio.
    
3. Дать среде загрузить зависимости и дождаться сборки Gradle.
    
4. Запустить приложение на эмуляторе или физическом устройстве с Android, нажав Run.
    

## Структура проекта

- `app/src/main/java/...` — исходный код приложения (экраны, логика, модели данных).
    
- `app/src/main/res` — ресурсы (разметка экранов, стили, строки, иконки).
    
- `AndroidManifest.xml` — основные настройки приложения и точка входа.
